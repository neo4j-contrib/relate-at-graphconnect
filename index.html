Relate! with your fellow graphistas

Like trading the strict schema of relational tables for the organic organization of a graph, the Relate! at GraphConnect unConference will guide community-driven participatory events during the conference. We'll provide demo space, organize hacking sessions, and host social discussions as you establish your own connections with other graphistas.

The un-conference is being non-planned using a github repository for collaboration and code sharing.

Share code using the github repository:

$ git clone https://github.com/neo4j-contrib/relate-at-graphconnect.git

Share ideas using the github wiki:

$ git clone https://github.com/neo4j-contrib/relate-at-graphconnect.wiki.git
